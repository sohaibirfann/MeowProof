Features

Everything you need.
Nothing you don't.

A focused utility that does one thing well — with a handful of extras for cat owners who've been burned before.

One-click keyboard lock

Blocks every key system-wide. Hit Lock Keyboard in the window, tray menu, or press Ctrl+Shift+L from anywhere.

Smart cat detection

Watches for paw-like typing in the background. Auto-locks the moment it spots cat chaos — no input needed from you.

Glassmorphism overlay

A full-screen frosted blur appears on every monitor when locked. Or use Stealth Lock for a completely silent, invisible lock.

Sound feedback

Optional sounds on lock and every blocked keystroke. Choose from 7 built-in sounds, or import your own. Great for deterring persistent paws.

Smart sleep behavior

Auto-unlocks on sleep or session lock — so you're never locked out of your own machine. Keeps your display awake while locked, too.

Launch at login

Set it and forget it — MeowProof starts automatically with Windows and repairs itself if you ever move it to a new folder.

Cat Detection

It knows when your
cat is typing.

MeowProof quietly watches for the telltale signs in the background. It never interferes with your own typing — no matter how fast you go.

Keys flying way faster than any human would ever type

Whole-paw pressing

Multiple keys held down all at once — like a paw resting on the keyboard

Total key chaos

A wild random mix of letters, numbers, and symbols — all in one go

Sitting on a key

One key held down for way, way longer than any person would
